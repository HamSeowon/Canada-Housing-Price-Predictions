# Canada-Housing-Price-Predictions
First Data science project predicting Canadian house prices from real estate listing data. 

- Dataset:
      https://www.kaggle.com/datasets/yuliiabulana/canada-housing
      44,896 listings, 23 raw features

  
- Task:
      predict `Price` from location, size, and amenity features by Regression

  
- Final model: Random Forest Regressor
      R² = 0.863 (5-fold CV), MAE ≈ $166,600
